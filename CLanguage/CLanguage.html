C Programming Language

History of C

The BCPL(Basic Combined Programming Language) is a computer programming language designed by Martin Richards, of Cambridge University in 1966.
Later 'B' was developed by Bell Labs. 'B' it is a programming language which has some complexities.
Then, Dennis Ritchie and Ken Thomson was improved B language at AT & T[telephone]Bell Laboratories in 1972.
They called the new language C and it can be used on various types of computers.
The programming language of 'C' was really conceived to be used on the operating system of UNIX.
It is compatible with many other operating system because it is one of the most widespread programming language.

What is C

  • 'C' language is a structured and procedural programming language.
    • It is a powerful and flexible language which can be used for a variety of applications, from business programs to engineering
  • 'C' language is particular a popular language for programmers of PCs(Personal computers) because it is relatively small; and it requires less memory than other languages.
  • Many versions of the UNIX-based opearting system are written in C.
  • It is high-level language
  • C language is much closer to the assembly language then the majority of the other high-level languages
    • An assembly language is a program written for a type of operating system which we cannot run on other operating system
