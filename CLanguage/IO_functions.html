1) Standard I/O Functions

A computer cannot make anything without giving any request. The user must give the request, the answer will be sent by computer.

  • The functions printf() and scanf() are used to perform input and output operations.
  • The printf() function prints the information provided on the screen (i.e. monitor).
  • The scanf() function receives the data from the user.
By using printf() and scanf() functions, we can print and input with any format with any format with the assistance of the format strings. Since the format strings decide only which type of data to enter or shown.
A limitations of using scanf() function does not allow empty spaces between the strings.

For example
when we enter a name Mother Theresa, a scanf() function allows only the string of "Mother".

The remainder of the word will not be stored in the respective variable. To avoid this kind of difficulty, we have to use a special function called gets().

Syntax

printf("<Format_String>,<List_of_Variables>");
scanf("<Format_String>,<& List_of_Variables>");

In this syntax, both functions are taken as two arguments, format string and list of variables.

  • <Format_String> : The format strings are used to determine the input or output data type.
  • <List_of_Variables> : The list of variables indicate variables which have data.

2) Character Input and Output

When you develop any application which is connected with entering information, it can have many numerical and character fields.

Character Input Functions
  • getchar() function :
    getchar() function allows the multiple characters but it assigns only onr character with the variable. for example : when you enter as "Rose" the variable receives only the "R".
  • getche() function :
    getche() function makes it possible only one character to enter by the user.
  • getch() function :
    getch() allows only character which is not visible once it is entered.

Character Output Functions

To carry out operations of output putch() and putchar() functions are used. Once that a character has been keyed, we can immediately show them with the assistance of the putchar() or putch() functions.
