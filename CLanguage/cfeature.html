Features of 'C'

Features of 'C' Language

  • 1) Operating system portability

    • We can write a C program in an Operating system and this program can be carried out in another Operating System.

  • 2) Highly efficient object code

    • This is a computer code which helps to execute our program quickly.

  • 3) Support of structured programming

    • The C language has a standard programming format. Each program must have the main function and other attributes.

  • 4) Standard function libraries

    • The C language is entirely integrated with the functions which are stored in the library.

  • 5) Ease of access to hardware features

    • C language supports writing programs to affect the hardware such as memory management, process scheduling etc.

The main reason to learn 'C' language

  • 1) Improved code readability

    • Compared with other language, C language has the instructions which are easily understood by the user. The compiler will program your code for a better speed.

  • 2) Dynamic register allocation

    • In C language, the storage allocation will be carried out during the execution time. Other automatic optimizations increase the total execution of code.
