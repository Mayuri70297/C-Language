Data Types


All the programming languages have data types which can be changed language with the language. The data types are essential for any computer programming language. Without them, it becomes very difficult to maintain information in a computer program. The data is a group of information which is not classified by category. It means that information will not be arranged in a suitable way. This is called raw data. Data can be numerical data or a character or data and so forth. These are called data types.


    Primary Data Types / Functional Types

  • Interger Data Type
    • The Interger Data Type allows only the values of integer without decimals but not characters. In this case, when you assign a character with a variable of interger, it will be regraded as alue of ASCII.
    • C has 3 classes of integer storage namely short int, int and long int.
    • All of these data types have signedi and unsigned forms.
    • A short intrequires half the space than normal integer values.
    • Unsigned numbers are always positive and consumes all the bits for the bits for tha mangnitude of the number. Therefore, the range of an unsigned integer will be from 0 to 65535.
    • The long integers are used to declare a longer range of values and it occupies 4 bytes of storage space.

  • Floating Point Data Type
    • The float allows numbers with the floating decimal points.
    • The float data type is used to store fractional numbers (real numbers) with 6 digits of precision.
    • When the accuracy of floating point number is insufficient, we can use the double to define the number.
    • The double is same as float but with longer precision and takes doubles space (8 bytes) than float.
    • To extend the precision further we can long double which occupies 10 bytes of memory space.
Type Size Value Range precision Format Specifier
float 4 bytes 1.2E-38 to 3.4E+38 6 decimal places %f
double 8 bytes 2.3E-308 to 1.7E+308 15 decimal places %lf
long double 10 bytes 3.4E-4932 to 1.1E+4932 19 decimal places %Lf

  • Character Data Type
    • The char type allows a character at the same time. A group of characters are called string.
    • Each character is stored in a separate place.
    • Character type variable can hold a single character and are declared by using the keyword char.
    • The Char Data type have signed and unsigned forms; both occupy 1 byte each, but having different ranges.
Type Size Value Range Format Specifier
char 1 bytes -128 to 127 or 0 to 255 %c
unsigned char 1 bytes 0 to 255 %c
signed char 1 bytes -128 to 127 %c
