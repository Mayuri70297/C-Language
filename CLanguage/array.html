Array


Array is a collection of homogenous data stored under unique name. The values in an array are called as 'elements of an array. " These elements are accessed by numbers called as 'subscripts or index numbers.

Arrays may be of any variable type. Array is also called as 'subscripted variable.

Array Declaration

To begin with, like other variables an array needs to be declared so that the compiler will know what kind of an array and how large an array we want.

  • An array is a collection of similar elements.
  • The first element in the array is numbered 0, so the last element is 1 less than the size of the array.
  • Before using an array its type and dimension must be declared.
  • Array elements are always stored in contiguous memory locations.
  • Total Size(in Bytes)
  • int a[3]
  • total size = length of array*size of data type



Types of Arrya

  • One Dimentional Array



    The array which is used to represent and store data in a linear form is called as 'single or one dimensional array.

  • Multi - Dimentional Array



    The array which has multiple subscript is known as multi-dimentional array.
