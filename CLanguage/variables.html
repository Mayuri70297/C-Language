Variables

The variables are generally defined as a name in a program. The smallest units in a program are known as variables Generally, the variables are used to store the values. A variable is a container for information which you want to store. Its value can change during the script. It should be unique. Variables must be declared before used.

Rules for Declaring Variables

  • The variable names are case sensitive. That means, small 'a' differs from capital 'A'.
  • The name of variable is a combination of alphabets, digits, or underscores.
  • The first character of the variable name must be an alphabet or an underscore.
  • No commas or white spaces are allowed in a variable name.
  • No special symbol other than an underscore can be used in a variable name.

Syntax to declaring varibles

Datatype variable_name [size];

In the Syntax, DataType denotes what type of information you are going to store in the variable. And, variable_name refers to the name of the variable. Size is an optional, which can be used as arrays.

Example :

char name[10];
int x=100;
